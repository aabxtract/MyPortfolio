# **App Name**: Hesed's Web3 Vista

## Core Features:

- Header Navigation: Sticky header with links to About, Projects, Experience, and Contact sections with smooth scrolling and underline hover animation.
- Hero Section: Prominent headline and subheadline introducing Hesed, with a call-to-action button to explore projects.
- Project Display: Minimal card layout showcasing personal projects with project names, descriptions, stack text, and links to live demos and repositories. Uses subtle scaling/border glow on hover.
- Skills List: Display a set of skills represented by plain text tags
- Experience Timeline: Compact vertical list of past experience, formatted as graphic designer, ui/ux designer, and web3 development intern
- Contact Information: Displays personal email, Github, LinkedIn, and X(Twitter) links

## Style Guidelines:

- Primary color: Bold violet (#A78BFA) for representing both creativity and Web3
- Background color: Deep gray (#0A0A0A) to support the dark theme and emphasize content
- Accent color: Soft blue (#5AA9E6) for subtle interactive highlights
- Headline font: 'Pixelify Sans' (sans-serif) for prominent display
- Body font: 'Inter' (sans-serif) as a system fallback font for comfortable reading
- Centered content layout with wide spacing and minimalist section separation.
- Subtle hover transitions and simple fade or slide-in animations on load.
